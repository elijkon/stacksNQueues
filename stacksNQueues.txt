Question 1: Stacking
Stack is first in last out.
-push(1)
We push 1.0 onto the stack. Stack = [1.0] (Top is 1.0)
-push(20)
We push 20.0 onto the stack. Stack = [1.0, 20.0] (Top is 20.0)
-pop()
This removes and returns the top item which is 20.0. Stack = [1.0] (Top is 1.0)
-push(pop() * 2)
First the pop function on the inside will be called which will return and remove 1.0. The stack is now empty.
It will then do the multiplication (1.0 * 2) which equals 2.0. Then we push the 2.0 onto the stack.
Stack = [2.0] (Top is 2.0)
-push(0)
We push the 0 onto the stack. Stack = [2.0, 0.0] (Top is 0.0)
-push(pop() / 2)
First the inner pop function executes which removes and returns 0.0. The stack is now [2.0]. Then we do the 
division (0.0 / 2 = 0.0). We then push 0.0. The stack ends up being [2.0, 0.0] (The top is 0.0).

Question 2: Queueing
Queueing is first in first out.
-push(2)
We push 2 into the queue. Queue = [2.0] (Top or head is 2.0)
-push(pop()+4)
We first run the pop() on the inside which removes and returns 2.0. We then do the addition
and do (2.0 + 4 = 6.0). We then push(6.0) onto the Queue. The Queue is now [6.0].
-push(5)
We push 5 into the Queue. Queue = [5.0,6.0] (Top is 6)
-push(pop()/2)
We first run the pop() inside of the push function. This removes and returns 6.0 from the Queue.
We then do the divison (6.0 / 2 = 3.0). We then push(3.0) into the Queue. Queue = [3.0, 5.0] (Top is 5.0).
-pop()
We pop() 5.0 from the Queue which returns it and removes it from our Queue. Queue = [3.0] (Top is 3.0).
-pop()
We pop() 3.0 from the Queue which returns it and removes it from our Queue. Queue = [] (It is empty).

Question 3: Find in Dequeue
def findInDeque(q, y):
    n = length of q
    left = 0
    right = n - 1

    while left <= right:
        
        if q[left] == y:
            return left
        
        if q[right] == y:
            return right
            
        left = left + 1
        right = right - 1

    return -1
